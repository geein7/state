# 키보드 단축키

## 일반 제어

| 단축키 | 동작 |
|--------|------|
| `Ctrl+C` | 현재 입력 또는 생성 취소 |
| `Ctrl+D` | Claude Code 종료 |
| `Ctrl+L` | 터미널 화면 지우기 |
| `Ctrl+O` | 상세 출력 토글 |
| `Ctrl+R` | 명령어 히스토리 역방향 검색 |
| `Ctrl+G` | 외부 텍스트 편집기에서 열기 |
| `Ctrl+F` | 모든 백그라운드 에이전트 종료 |
| `Ctrl+B` | 실행 중인 작업 백그라운드로 |
| `Ctrl+T` | 작업 목록 표시 토글 |
| `Ctrl+V` / `Cmd+V` / `Alt+V` | 클립보드에서 이미지 붙여넣기 |
| `Esc` + `Esc` | 되감기 또는 요약 |

## 텍스트 편집

| 단축키 | 동작 |
|--------|------|
| `Ctrl+K` | 줄 끝까지 삭제 |
| `Ctrl+U` | 전체 줄 삭제 |
| `Ctrl+Y` | 삭제된 텍스트 붙여넣기 |
| `Alt+Y` | 붙여넣기 히스토리 순환 (Ctrl+Y 후) |
| `Alt+B` | 한 단어 뒤로 커서 이동 |
| `Alt+F` | 한 단어 앞으로 커서 이동 |

## 권한 & 모드

| 단축키 | 동작 |
|--------|------|
| `Shift+Tab` 또는 `Alt+M` | 권한 모드 전환 |
| `Option+P` (macOS) / `Alt+P` | 모델 전환 |
| `Option+T` (macOS) / `Alt+T` | 확장 사고 토글 |

## 커스터마이즈

키 바인딩은 `~/.claude/keybindings.json` 에서 수정하거나 `/keybindings` 명령어로 관리할 수 있습니다.
