# IDE 통합

## VS Code 확장

- **기능**: 프롬프트 박스, 파일 참조, 세션 재개, 멀티 대화, git 통합
- **파일 참조**: `@` 으로 파일/폴더 언급
- **Git**: `/git` 으로 커밋 및 PR 생성
- **설정**: VS Code 설정에서 구성

## JetBrains IDE 플러그인

지원 IDE: IntelliJ, PyCharm, WebStorm 등

- **기능**: 터미널 통합, 파일 컨텍스트, 외부 터미널 접근
- **WSL**: Windows Subsystem for Linux 특별 설정 지원

## Chrome 브라우저 확장

```bash
# 활성화
claude --chrome
```

- **기능**: 웹 자동화, 폼 입력, 데이터 추출, 디버깅
- **명령어**: 웹 앱 테스트, 페이지 상호작용, 스크린샷
- **권한**: `/chrome` 으로 사이트 권한 관리

## 데스크탑 앱

- **기능**: 통합 UI, 원격 세션, 예약 작업, 플러그인
- **환경**: 로컬, 원격, SSH 세션
- **미리보기 서버**: 변경사항 자동 검증
- **협업**: 팀원과 공동 작업 (Cowork)

## 웹 버전

- **접근**: claude.ai
- **기능**: 클라우드 실행, diff 보기, 환경 설정
- **보안**: 제한 도메인 목록
- **지속성**: 웹/터미널 간 세션 이동 가능
