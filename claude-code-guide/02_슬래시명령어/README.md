# 슬래시 명령어

## 설정 & 정보

| 명령어 | 설명 |
|--------|------|
| `/config` | 설정 인터페이스 |
| `/agents` | 서브에이전트 관리 (생성, 편집, 삭제) |
| `/help` | 사용 가능한 명령어 목록 |
| `/doctor` | 문제 진단 |
| `/theme` | 출력 테마 변경 |
| `/vim` | Vim 편집기 모드 활성화 |
| `/terminal-setup` | 터미널 설정 |

## 세션 관리

| 명령어 | 설명 |
|--------|------|
| `/resume` | 이전 세션 재개 |
| `/clear` | 새 세션 시작 |
| `/rename` | 현재 세션 이름 변경 |
| `/memory` | CLAUDE.md 및 자동 메모리 보기/편집 |
| `/init` | CLAUDE.md 초기화 |
| `/compact` | 대화 기록 압축 |

## 권한 & 보안

| 명령어 | 설명 |
|--------|------|
| `/permissions` | 권한 규칙 보기/관리 |
| `/add-dir` | 접근 가능 디렉토리 추가 |
| `/mcp` | MCP 서버 설정/인증 |
| `/hooks` | 훅 보기/관리 |

## 유틸리티

| 명령어 | 설명 |
|--------|------|
| `/plugin` | 플러그인 설치/관리 |
| `/cost` | API 비용 추적 |
| `/btw` | 빠른 부가 질문 (도구 없음) |
| `/loop` | 반복 작업 예약 |
| `/keybindings` | 키보드 단축키 커스터마이즈 |
